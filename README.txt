# Loja Virtual 🛒

## Descrição do Projeto 📝
Este é um projeto de loja virtual desenvolvida para a disciplina de WEB 1 usando Django como framework principal. A aplicação permite que clientes navegem por produtos e realizem compras de forma simples e segura.

## Funcionalidades ✨
- 🛍️ Navegação fácil entre categorias de produtos
- 🛒 Carrinho de compras completo
- 💳 Sistema de pagamentos seguro
- 📦 Gerenciamento de pedidos
- 👨‍💼 Painel de administração para gerenciar produtos e pedidos
- 🔐 Autenticação de usuários

## Tecnologias Utilizadas 🛠️
- **Backend:** Python com Django
- **Frontend:** HTML, CSS, JavaScript
- **Banco de Dados:** SQLite
- **Servidor:** Django Development Server

## Como Começar 🚀
1. Clone o repositório: `git clone https://github.com/Alvaro-Sousa/Loja_Virtual`
2. Navegue até a pasta: `cd Loja_Virtual`
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute as migrações: `python manage.py migrate`
5. Inicie o servidor: `python manage.py runserver`
6. Acesse em `http://localhost:8000`
